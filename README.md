# Mathematical Theory of Spatiotemporal THz Electric-Field Detection

This repository contains a theory-only GitHub Pages document that begins with the optical delay line and develops the complete mathematical chain leading to the reconstructed field cube `E(x,y,t)` and its frequency-resolved vector and Stokes representation.

## Files

- `index.html` — complete mathematical theory page
- `README.md` — repository description and deployment instructions

## Publish with GitHub Pages

1. Create a **public** GitHub repository named `thz-temporal-evolution-theory`.
2. Upload `index.html` and `README.md` to the repository root.
3. Open **Settings → Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select **main** and **/(root)**, then save.

The site address will be:

`https://kararwaldeepak.github.io/thz-temporal-evolution-theory/`
