# Temporal Evolution of a Terahertz Electric Field

A theory-only GitHub Pages document describing:

- optical delay-line sampling;
- delay-step conversion to time;
- temporal and spectral sampling limits;
- construction of \(E_x(x,y,t)\) and \(E_y(x,y,t)\);
- step-by-step temporal evolution of a broadband THz transient;
- pixel-wise Fourier transformation;
- complex amplitude and phase;
- narrowband time reconstruction;
- Stokes parameters and polarization;
- ROI averaging, timing blur, and uncertainty.

## Publish

1. Create a public GitHub repository named `thz-field-theory-general`.
2. Upload `index.html` and `README.md`.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Select `main` and `/ (root)`.
6. Save.

The page address will be:

`https://kararwaldeepak.github.io/thz-field-theory-general/`
