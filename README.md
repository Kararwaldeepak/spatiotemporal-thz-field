# Spatiotemporal Detection of THz Electric Fields

Educational GitHub Pages website about spatiotemporal and polarization-resolved mapping of broadband terahertz electric fields.

## Publish

1. Open **Settings**.
2. Open **Pages**.
3. Select **Deploy from a branch**.
4. Choose `main` and `/ (root)`.
5. Click **Save**.

Expected URL:

`https://kararwaldeepak.github.io/spatiotemporal-thz-field/`
